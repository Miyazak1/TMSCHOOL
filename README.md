# TMPS Private High School Static Website

This is a no-backend, no-WordPress static website for an OSSD school.

Open `index.html` in a browser to preview the site. The course catalogue is driven by local data in `script.js`, and the contact page uses direct phone and email links instead of a backend form.

On Windows, double-click `start-local.bat` to start a local preview server and open the homepage.

The first version includes these static pages:

- `index.html`
- `courses.html`
- `admissions.html`
- `international-students.html`
- `about.html`
- `contact.html`

## Files

- `index.html` - homepage structure and content
- `courses.html` - filterable OSSD course catalogue
- `admissions.html` - application and course planning flow
- `international-students.html` - international student information
- `about.html` - school positioning and learning approach
- `contact.html` - admissions contact page
- `start-local.bat` - Windows local preview launcher
- `styles.css` - responsive visual design
- `script.js` - course filters, course details, and FAQ accordion
- `docs/static-website-plan.md` - planning document and reference-site inheritance strategy
